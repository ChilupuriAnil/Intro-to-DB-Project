Fd

Moves the turtle forward by desired pixels
           syntax: fd 40

Bk

Moves the turtle backward by desired pixels
           snytax: bk 40

Rt

Turns the turtle right side by desired angle
           snytax: rt 90

Lt

Turns the turtle left side by desired angle
           snytax: lt 90

Repeat

Repeats any number of commands any number of times
      snytax: repeat 4 [fd 40 rt 90]

Pu

           snytax: pu

Pd

Pd command is to counter pu command
           snytax: pd

Pc

Used to change the color of lines
           snytax: pc 1

Hc

Hc command is used to ide the cursor
           snytax: hc
